Button

Button styles can be applied to any element. Typically you'll want to use either a <button> or an <a> element:

Link with .btn class Primary button Disabled button 
<button class="btn">Simple button</button>
<a class="btn" href="http://wearecube.ch">Link with .btn class</a>
<a class="btn is-primary">Primary button</a>
<a class="btn is-disabled">Disabled button</a>

Button sizes

Three sizes for the buttons: small, medium (default) and big.

Big button Big button 
<a class="btn is-small">Big button</a>
<a class="btn is-big">Big button</a>

Forms

The HTML fieldset element is used to group several controls as well as labels (label) within a web form.

Login 
<form action="" method="">
  <fieldset>
    <legend>Login</legend>
    <label for="email">Email</label> <input id="email" name="email" type="email" placeholder="type your email here">
    <label for="password">Password</label> <input id="password" name="password" type="password" placeholder="type your password here">
    <input id="check" type="checkbox" name="check"> <label for="check">I'm ok</label>
  </fieldset>
</form>

Headings

Heading elements implement six levels of document headings, <h1> is the most important and <h6> is the least. A heading element briefly describes the topic of the section it introduces. Heading information may be used by user agents, for example, to construct a table of contents for a document automatically.
